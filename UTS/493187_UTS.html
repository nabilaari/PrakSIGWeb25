    Pertemuan 2 dan 3: HTML Sederhana

    HTML (Hypertext Markup Language), merupakan bahasa standar dalam menyusun halaman web. HTML dapat memuat informasi mengenai gambar, text, table, ataupun menyisipkan hyperlink supaya dapat menuju ke halaman web yang lain.

    Pertemuan 4: CSS dan Javascript

    CSS merupakan bahasa pemrograman yang berfungsi untuk memisahkan konten yang ditampilkan dari halaman web yang dibuat, selain itu dengan penggunaan CSS, pengguna dapat mengatur tampilan halaman web. Sedangkan Javascript merupakan bahasa pemrograman yang dapat digunakan supaya halaman web yang digunakan menjadi lebih interaktif.

    Minggu 5: Bootsrap

    Bootstrap merupakan salah satu bentuk framework dengan bahasa pemrograman javascript. Melalui bootstrap, operator dapat melakukan editing sederhana mengenai styling yang digunakan pada halaman web

    Minggu 6: Pembuatan Peta Leaflet

    Leaflet merupakan salah satu WebMap API yang dapat digunakan untuk menambahkan peta ke dalam halaman web.

    Minggu 6 dan 7: Pembuatan Peta Leaflet-GeoJSON

    Pada pertemuan ke-7, terdapat penambahan materi mengenai penggunaan GeoJSONdalam pembentukan peta yang akan ditampilkan pada halaman web. Operator dapat menambahkan data spasial dengan format GeoJSON sebagai layer baru, seperti adanya penambahan data batas wilayah.
